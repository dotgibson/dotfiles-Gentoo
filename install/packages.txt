# mise (not in the tree) and tree-sitter-cli (cargo) are handled by bootstrap.sh.
#
# Heads-up: several modern Rust CLIs may be ~amd64 (testing-keyworded) on a
# stable profile. If an emerge is blocked by keywords/USE, see
# gentoo/package.accept_keywords.example and gentoo/package.use.example.
# Building from source is slow — enable the binhost (--getbinpkg, see README).
